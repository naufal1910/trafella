## Tech stack

This stack is selected to support a high-performance, single-page application (SPA) as requested.

### Framework & Runtime
- **Application Framework:** [Next.js (for React SPA & API routes)]
- **Language/Runtime:** [Node.js]
- **Package Manager:** [npm]

### Frontend
- **JavaScript Framework:** [React]
- **CSS Framework:** [Tailwind CSS]
- **UI Components:** [shadcn/ui]

### Database & Storage
- **Database:** [PostgreSQL (for POI database)]
- **ORM/Query Builder:** [Prisma]
- **Caching:** [Redis (for caching expensive itinerary lookups)]

### Testing & Quality
- **Test Framework:** [Jest]
- **Linting/Formatting:** [ESLint, Prettier]

### Deployment & Infrastructure
- **Hosting:** [Vercel (ideal for Next.js/React SPAs)]
- **CI/CD:** [GitHub Actions]

### Third-Party Services
- **Authentication:** [NextAuth.js (if user accounts are added)]
- **Email:** [SendGrid]
- **Monitoring:** [Sentry]
